# Employee Project Analysis (Python)

This project analyzes employee and project data using Python, Pandas, and NumPy.

## Key Tasks
- Created multiple DataFrames for employee, project, and seniority data
- Handled missing project cost values using running average
- Performed data cleaning and transformation
- Merged datasets for consolidated analysis
- Calculated employee bonuses based on project completion
- Adjusted designation levels based on project performance
- Aggregated total project cost per employee

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
