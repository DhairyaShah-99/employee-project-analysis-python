# 📊 Employee Project Analysis (Python)

## 🧩 Problem Statement

Organizations often struggle to track employee performance, project costs, and efficiency across multiple teams. This project aims to analyze employee and project data to uncover insights related to performance, cost allocation, and bonus distribution.

---

## 📁 Dataset Description

The dataset consists of multiple tables:

* **Employee Data**: Employee ID, Name, Designation
* **Project Data**: Project ID, Cost, Completion Status
* **Seniority Data**: Employee levels and hierarchy

These datasets were combined to perform a consolidated analysis.

---

## 🎯 Objective

* Analyze employee contribution to projects
* Handle missing cost data effectively
* Calculate performance-based bonuses
* Evaluate employee efficiency and designation changes
* Aggregate total project cost per employee

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Jupyter Notebook

---

## 🔧 Data Processing & Approach

### 1. Data Cleaning

* Handled missing values in project cost using **running average technique**
* Ensured consistency across datasets before merging

### 2. Data Transformation

* Created multiple DataFrames for structured analysis
* Merged employee, project, and seniority data

### 3. Feature Engineering

* Calculated **employee bonuses** based on project completion
* Adjusted **designation levels** based on performance metrics

### 4. Aggregation

* Computed **total project cost per employee**
* Evaluated contribution across multiple projects

---

## 📊 Key Insights

* Employees working on multiple projects contributed significantly more to total cost allocation
* Missing cost values, when handled properly, prevented skewed analysis
* Performance-based bonus calculation highlighted high-performing employees
* Certain employees showed mismatch between designation and actual performance

---

## 📈 Business Impact

* Helps management identify **high-performing employees**
* Enables **fair bonus distribution** based on contribution
* Supports **data-driven promotion decisions**
* Improves visibility into **project cost allocation**
